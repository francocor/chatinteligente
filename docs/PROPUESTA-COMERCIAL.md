```

6. Bot: Muestra horarios disponibles

8. Bot: Confirma y deriva a agente

```

### Escalamiento a Humano

El sistema detecta automáticamente cuando debe escalar:

- **Por关键词**: "hablar con alguien", "urgencia", "no entiendo"
- **Por sentimiento**: Frustración detectada (análisis de sentimiento)
- **Por retries**: Después de 2 intentos fallidos
- **Por configuración**: Ciertas consultas van siempre a humano
